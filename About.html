About Travel Japan


Origin of the Website Project

This website was further developed using Visual Studio Code as part of an English subject project. It applies knowledge in web development and digital communication. The concept of this project originated from the desire to create an online platform in the form of a website that provides informative content and is also practically functional.



Outcome of the Website Project

Through the continued development using Visual Studio Code, the Travel Japan website was created. This project not only contains informative content but also provided valuable experience in teamwork. In addition to enhancing collaboration skills, the website serves as a useful tool for future users and demonstrates the effectiveness of experiential learning and the ability to apply prior knowledge to develop a functional and informative platform like Travel Japan.



Issues Encountered During Website Development

1.Database Connection Issue: The database could not be connected successfully.

2.Web Page Display Issue: The webpage did not display the submitted data as expected.

3.Incomplete Data Transfer: When transferring files to another device, some data files were found to be missing or incomplete.

4.Server Activation Requirement: In order for the website to function properly, it requires the developer or the device hosting the website files to manually activate the server (also known as enabling the live server) so that the data can be displayed and accessed.
